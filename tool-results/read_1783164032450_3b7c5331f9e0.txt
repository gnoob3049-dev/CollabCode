   611→## Audio Notifications System (Task 6-c)
   612→
   613→### Overview
   614→Implemented a complete audio notification system for CollabCode using the Web Audio API, providing short, pleasant sound feedback for key events without requiring any external audio files.
   615→
   616→### Part 1: Audio Utility (`src/lib/audio.ts`)
   617→- Created `SoundType` union type: `'message' | 'success' | 'error' | 'join' | 'leave' | 'save' | 'run_start' | 'run_complete'`
   618→- `playSound(type, options?)` — generates sounds programmatically using `OscillatorNode` + `GainNode`:
   619→  - **message**: Two ascending tones (800Hz → 1200Hz, 100ms each)
   620→  - **success**: C-E-G arpeggio (523Hz, 659Hz, 784Hz, 80ms each)
   621→  - **error**: Low descending sawtooth buzz (300Hz → 200Hz, 200ms)
   622→  - **join**: Pleasant 880Hz sine chime with gentle attack/decay (200ms)
   623→  - **leave**: Soft descending tone (600Hz → 400Hz, 200ms)
   624→  - **save**: Quick 1000Hz sine blip (50ms)
   625→  - **run_start**: Short noise-burst click via high-freq square wave (30ms)
   626→  - **run_complete**: Delegates to `success` or `error` based on `hasErrors` option
   627→- `isAudioEnabled()` / `setAudioEnabled(bool)` / `toggleAudio()` — persists to `localStorage` key `collabcode-audio-enabled` (default: enabled)
   628→- Lazy `AudioContext` initialization — only created on first user interaction, auto-resumes if suspended
   629→

   631→- **EditorTopBar.tsx**: Added `Bell`/`BellOff` icon button before the Settings button with tooltip showing on/off state; filled `Bell` when enabled, outline `BellOff` when disabled; dimmed color when disabled
   632→- **EditorStatusBar.tsx**: Added small bell icon in the right section before the Live indicator with matching tooltip; consistent visual indicator
   633→
   634→### Part 3: Event Sound Integration (EditorPage.tsx)
   635→- **Chat message received** (panel not open): plays `message` sound + sonner toast with sender name and 50-char preview, auto-dismiss 4s, "Open chat" action opens panel
   636→- **User joined**: plays `join` chime
   637→- **User left**: plays `leave` tone
   638→- **Code run started**: plays `run_start` click
   639→- **Code run completed (success)**: plays `run_complete` (delegates to `success`) + toast "Code executed successfully"
